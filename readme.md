## Setup Instructions

Terminal 1
1. `git clone https://github.com/mounvip1525/QuoVit-Backend.git` 
2. `npm install`
3. `node start`

Terminal 2
1. `git clone https://github.com/mounvip1525/QuoVit.git` 
2. `cd frontend`
3. `npm install & npm start`

. The app is now running at http://localhost:3030/landing
